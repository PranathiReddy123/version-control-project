Version Control Project (EduSkills Web Development)

Steps Performed:
1. git init                - Initialized a new git repository.
2. git add index.html     - Added index.html to staging.
3. git commit -m "Initial commit" - Committed the file.
4. git status              - Checked the current status.
5. (Optional) git log      - Viewed commit history.

This project is a basic HTML page used to demonstrate git version control.
